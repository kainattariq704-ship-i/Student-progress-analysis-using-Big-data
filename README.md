# Student progress analysis-using Big-data using Machine Learning

## Overview

This project predicts student academic performance using Machine Learning techniques. The dataset contains demographic, educational, and behavioral information about students. Various machine learning models are evaluated and compared to identify the most effective approach for predicting student performance.

## Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature importance analysis using Random Forest
* Feature selection
* Model comparison using Cross Validation
* Hyperparameter tuning with GridSearchCV
* Ensemble learning techniques
* Performance evaluation using Mean Squared Error

## Dataset

The dataset contains information about students including:

* Gender
* Nationality
* Grade Level
* Educational Stage
* Number of Raised Hands
* Discussion Participation
* Resource Visits
* Absence Days
* Parent Satisfaction
* Survey Responses

Target Variable:

* Class (Low, Middle, High)
