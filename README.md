# Student Progress Analysis using Big Data

A Big Data project analyzing student progress data using Python and Pandas. This project processes student datasets, performs data analysis, and generates insights about student performance trends.

## Features
- Analyze student performance data using Python.
- Handle large datasets efficiently.
- Generate meaningful insights from student progress data.

## Technologies Used
- **Python** – For data analysis and processing.
- **Pandas** – For data manipulation and analysis.
- **CSV** – Dataset storage.
- **Chromedriver** – For any web automation or scraping tasks (if used).

## Dataset
- The dataset is provided in `datasets.csv`.
- Contains student performance data for analysis.
